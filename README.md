# joolomee.com

Portfolio & services website for **Joana Lopes Mesquita** — Full Stack Designer & Creative Director.

**Stack:** Next.js 14 (App Router) · TypeScript · Tailwind CSS · Framer Motion · Lenis · Vercel

## Quick start

```bash
npm install
npm run dev        # http://localhost:3000
npm run build && npm start
```

## Deploy

```bash
vercel --prod
```

Then in the Vercel dashboard add the custom domain **joolomee.com**.
